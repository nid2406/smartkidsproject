HTML

HTML stands for Hypertext Markup Language, and it is the most widely used language to write Web Pages.

• Hypertext refers to the way in which Web pages are linked together.
• Markup Language means you simply "mark-up" a text document with tags that tell a Web browser how to structure it to display.

Basic HTML Tags

Tag Description
<!DOCTYPE...> This tag defines the document type and HTML version.
<html> This tag encloses the complete HTML document.
<html lang="en"> The lang attribute allows you to indicate the main language used in a document.
<head> This tag represents the document's header which can keep other HTML tags like <title>, <link> etc.
<title> The <title> tag is used inside the <head> tag to mention the document title.
<body> This tag represents the document's body which keeps other HTML tags like <h1>, <div>, <p> etc.
<h1> This tag represents the heading 1.
<h2> This tag represents the heading 2.
<h3> This tag represents the heading 3.
<h4> This tag represents the heading 4.
<h5> This tag represents the heading 5.
<h6> This tag represents the heading 6.
<p> This tag represents a paragraph.
<br> This tag inserts a single line break.


Except few tags, most of the tags have their corresponding closing tags.

Meta Tags

The <meta> tag is used to provide some additional important information. This tag is an empty element and so does not have a closing tag but it carries information within its attributes.
You can include one or more meta tags in your document based on what information you want to keep in your document but in general, meta tags do not impact physical appearance.

• Character set
<meta charset="UTF-8"/>
• Compatible with browser
<meta http-equiv="X-UA-Compatible" content="IE=edge" />
• View on different screen
<meta name="viewport" content="width=device-width, initial-scale=1.0" />

• Document Refreshing
<meta http-equiv="refresh" content="5" />

• Page Redirection Character set
<meta http-equiv="refresh" content="5; url=http://www.google.com" />



The text editor we use to write our page is Notepad and to see our page, we need any kind of browser.
