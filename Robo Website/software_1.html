Software
RoboAnalyzer, apart from providing visualization and simulation features, can also analyze the path along which it moves. This helps in better understanding of the problems faced in the movement along the path like collisions taking place, the force and torque requirements etc. It provides us with the coordinates of the trajectory, for future applications. In addition, it computes and plots the motion parameters (velocity and acceleration) of the arm at any given time. The software is developed to do both kinematic and dynamic analysis including options for analysis under gravity. This software is based upon one of the advanced recursive robot dynamics algorithms developed at IIT Delhi in last 15 years. Moreover, till now no software provides the ability to analyze the trajectory along which robot moves. So, this provides better understanding to industry experts and academicians. This ultimately leads the company in saving a lot of costs involved in various phases of a system's life-cycle.
The life-like simulation behavior provided by this module helps in analyzing system performance in fine detail. This is a good visualization tool which can be used by academic institutes to clearly teach the concepts of robotics as well as in industry to train individuals. The software's interactive plotter allows easy understating of robot dynamics with time. This, on one side, helps students and academicians to better understand and visualize robots' movements along the path it moves; on the other side, it helps industry experts in better analyzing that path. So, it has multifaceted benefits.

Some of its striking features are:
    • Ability for kinematic and dynamic analysis.
    • Plotting of resulting data into various charts - Pie Chart, Bar Chart, Line Chart.
    • Preprocessing and post processing applications so as to integrate with commonly used robotic systems.
    • A user friendly simulation viewer.
    • Microsoft compatible installer.
    • Providing stability features for better performance in new operating systems.
