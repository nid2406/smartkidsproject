Introduction of Web Designing

Web page
The information on the internet in the form of digital pages called web pages. It can have text, images, videos, any combination… etc.

Website
A collection of publicly accessible, interlinked web pages. There are thousands and thousands of websites are present.

  • Social website to chat with the Friends and families,
  • Photo websites to Share photos,
  • Travel websites to book Trips and book tickets,
  • News websites to see what is going on in the World,
  • Ecommerce websites to purchase things,
  • Entertainment websites like YouTube,
  • Education websites, which give us courses,
  • Health Websites,
  • Fitness website,
  • Information websites,
  • Banking Websites, Etc.
Web Browser
A program that allows you to access the entire information available on the web. Different types of web browsers are:-
  • Google chrome
  • Apple Safari
  • Opera
  • Firefox
  • Internet explorer. Etc.
Web or World Wide Web
A system by which users can access all the websites and documents and other web resources to be accessed over the Internet.
Web programming is a method which can be used to design and build various web pages. There are three web programming languages
  1. HTML
  2. CSS
  3. Javascript.
HTML
Is the language in which we make basic structure of our web page.
CSS
Is the language we use to beautify our webpage.
JS or javascript
Is the language by which we make our web page more interactive.
Without these languages, we cannot make any kind of web page or website. If I want to make any website, we have to use HTML. And CSS and javascript depending on what we want, to do.
HTML and CSS are the Front-end languages. And JavaScript is Back End Language.

The text editor we use to write our page is Notepad and to see our page, we need any kind of browser.
